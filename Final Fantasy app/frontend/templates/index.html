Project Overview

Fantasy Sports Team Analyzer is a Windows desktop application that evaluates fantasy rosters using live player statistics and league scoring rules, providing objective guidance for trades and roster balance.

  • Inputs: Roster, league settings, trade proposals, live player stats (sports API).
  • Outputs: Team Strength score, Position Breakdown, Trade Evaluation & recommendations, exportable reports.
  • Tech: Python (Flask), SQLite, HTML/CSS/JS in a native window via pywebview.
