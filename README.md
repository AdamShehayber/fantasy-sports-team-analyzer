# Fantasy Sports Team Analyzer

AI-assisted fantasy football analyzer that evaluates team strength, trade value, and roster decisions using projections, custom scoring algorithms, and a relational database.

## Features
- Team strength analysis
- Trade evaluation
- Player projections
- AI-assisted recommendations
- SQLite database

## Tech Stack
- Python
- Flask
- SQLite
- SQLAlchemy
- PyWebView
